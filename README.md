## 五孔计算器

专业的五孔计算工具，支持多种模板和组合计算。

## 功能特点
- 📐 多种主轴/下轴模板
- 🔄 多模板组合功能
- 📊 实时图形展示
- 🎬 视频教程指导
- 🧮 反向计算

## 使用方法
1. 选择模板或自定义点
2. 自动计算扇形角度和直径
3. 导出计算结果

## 视频教程
点击"视频教程"按钮查看详细使用教程，学习如何：
- 加载和使用模板
- 进行多模板组合
- 使用递增计算器
- 导出程序文件
